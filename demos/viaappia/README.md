# General idea of the project

The [Via Appia](https://en.wikipedia.org/wiki/Appian_Way) was the main highway connecting ancient Rome to Greece and the East. Its total length was more than 500 km running al the way to modern-day [Brindisi](https://www.google.nl/maps/place/Brindisi+BR,+Italy/@40.6422249,17.9009354,7z/data=!4m2!3m1!1s0x13467a3bc980ec6d:0x110cef7cc03daf9). 

Over time, many funeral monuments were erected along the street. Currently there is an effort underway to catalogue all of these monuments
